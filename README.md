### Basic system specs:
- Ruby - 2.2.0
- Rails  4.2.0

### DB adapter:
- PostgreSQL

### Configuration
* bundle
* copy config/database.example.yml into config/database.yml
* set your username and password into config/database.yml for postgreSQL
* rake db:create
* rake db:migrate
* rails s
